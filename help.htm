Convert Pro Documentation

What is Convert Pro?

Convert Pro is a simple to use programme that converts units, for example metres to feet.

How do I use it?

  1. Select the conversion type from the Conversion drop-down.
  2. Enter the value you want to convert in the From text box.
  3. Select the unit that the value is in from the From drop-down.
  4. Select the unit you want to convert to from the To drop-down.
  5. Press Convert.
  6. The converted value is displayed in the To text box.

What conversions are there?

Convert Pro is shipped with the following conversion types:

Length
  • mm (millimeters)
  • cm (centimeters)
  • m (meters)
  • km (kilometers)
  • in (inches)
  • ft (feet)
  • yd (yards)
  • mi (miles)
Weight
  • mg (milligrams)
  • g (grams)
  • kg (kilograms)
  • oz (ounces)
  • lbs (pounds)
  • stone
Temperature
  • C (Celsius)
  • F (Fahrenheit)
  • K (Kelvin)
Speed
  • mph (miles per hour)
  • km/h (kilometers per hour)
  • m/s (meters per second)
  • kts (knots)
Time
  • Seconds
  • Minutes
  • Hours
  • Days
  • Weeks

How do I create/modify conversion types?

Pressing Edit on the main form will open the edit window. From here you can do two things.

  • Modify and add to existing conversion types:
    Select the conversion type to edit. Click on a unit to edit it and press Edit. Make any changes in the text boxes on the right then press Apply. (For help understanding the text box meanings see the next section.) To cancel changes press Cancel.
    To add a new unit click on <new> in the units list. Enter the values for this unit in the text boxes on the right and press Apply.
  • Add a new conversion type:
    Press New and type the name of the new conversion in the input box, press OK. Add new units using the method described above.

Once you have finished editing press OK to save settings and exit the edit window. Or press cancel to exit the edit window without saving any changes.

What are relations and offsets?

When making new units it is important to understand what is meant by relation and offset. The way that Convert Pro works is to first convert your value to a base unit and then convert that new value to your selected unit. The base unit's relation is always 1. So in the case of length mm is the base unit and cm has a relation of 10 i.e. there are 10 mm to the cm. When creating a new conversion it is suggested that you use the smallest unit as the base.
The offset is zero most of the time but sometimes it must be used. Any time 0 of one unit does not equal 0 of another e.g. 0ºC = 32ºF offset in this case is 32 for Fahrenheit. The offset is used shunt the values back and forward to allow conversion.
